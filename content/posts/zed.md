# My thoughts on Zed

- OpenCode ACP
    - For bash command only summaries are visible
    - After restarting OpenCode, all non-tool-call messages are gone

Claude Code
- Only possible in Terminal threads
- would be nice if terminal threads could scan their content to have indicators

Remote servers
- very little indication if anything is happening

Agent window

On two-line changes with a horizontal scrollbar, the scrollbar covers part of
the change. 

When creating a new agent session, the plus button menu is not searchable
- there are dozens of agents in the registry, this is unwieldly without search
or frecency


Agent itself
- does not support querying available models from a server
- No GLM or MiniMax providers
- Does not support OpenCode free models
